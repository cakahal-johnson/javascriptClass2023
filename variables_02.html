Javascript||Variables

Variables are like a memory container for storing data value either words or numbers

Relus of making or assigning a Variables:

  1. Names can contain letters, digits, underscores, and dollar signs
  2. Names must begin with a letter
  3. Names can also begin with a $ and _ but not with a number
  4. Name are very case sensitive (Name and name or nAme all is a different variable name)
  5. Reserved words (like Javascript keyword) cannot be used as a name variables which is primitive datatype

    6. All variables must be identified with a unique name called identifiers

    Javascript uses = sign to assign a varibales as an identifiers, this == (operators sign) is not same with = (assigning varibales)
