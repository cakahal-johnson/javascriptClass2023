Browser Object Model(BOM) while Document Object Model(DOM)

when a web page is loaded, the browser creates a Document Object Model of the page

  • Document
  • Root element: ( < HTML > )
  • Root element has: < head > and < body >
  • Head element has: < title > and < Text >, < meta >
  • Body element has: attribute < a > and < h1 >

with this Object Model javascript creates a dynamic HTML

  1. it can change all the HTML elements in the page
  2. it can change all the HTML attribute in the page
  3. it can change all the CSS style in the page
  4. it can remove existing HTML elements and attribute in the page
  5. it can add new HTML elements and attribute in the page
  6. it can react to all existing HTML events in the page
  7. it can create new HTML events in the page

DOM Interface

  • The HTML element as objects
  • The properties of all HTML elements
  • The methods to access all HTML elements
  • The events of all HTML elements

BOM categories

  1. js window: window.open() to open a new window, window.close(), window.moveTo(), window.resizeTo()
  2. js screen: screen.width, screen.height, availWidth, availHeight, colorDepth (24bits = True color, 32bits (Deep color)) , pixelDepth

  3. Location(URL): has href(current URL page), hostname (domain name of web host), pathname(current page file path and filename), protocol(returns the web protocol used(http: or https)), assign() loads a new document

  4. History: contains the browsers History it has history.back() means same as clicking back in the browser and same as forward()
  5. Navigator: this object contains information about the visitor's browser, properties: navigator.appName, navigator.appCodeName, navigator.platform

  6. Js Popup Alert
